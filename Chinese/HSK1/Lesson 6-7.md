## LESSON 7

Pronombre personal + 的(de) $\rightarrow$ Pronombre prosesivo

的(de) indica pertenencia, Ejemplo:
- 中国的(zhōn guó de) : De china
- Jesus de : De jesus

| Español | Pinyin | Caracter |  |
| ---- | ---- | ---- | ---- |
| mio, mia, mios, mias | wǒ de | 我的 |  |
| tuyo, tuya | nǐ de | 你的 |  |
| Suyo, suya | nín de | 您的 |  |
| suyo, suya  | tā de | 她的 |   |
| suyo, suya | tā de | 他的 |    |
